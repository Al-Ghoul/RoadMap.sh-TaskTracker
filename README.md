# Task tracker

Task tracker is a project used to track and manage your tasks.

## Development [Nix]
This project needs [Zig](https://ziglang.org/) 13.0.0
```bash
# Clone & cd into project
# Enter development shell
nix develop --impure

# Run zig build
zig build
```

## Technology stack
- [Zig](https://ziglang.org/)
- [Nix](https://nixos.org/nix/)
